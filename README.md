## Проект

- реализует базовые классы для описания товаров и категорий.

## Какие технологии использованы?

- Python (версия >= 3.10)
- Poetery (для управления зависимостями)
- Pytest (для написания тестов)
- Flake8 (для соблюдения стандартов оформления кода)
